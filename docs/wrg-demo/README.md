# WRG Demo — Documentation

This folder contains all documentation, flows, scripts, and reference material for the **WRG demo** version built on the `demo` branch of UI-Dealer.

## Structure

```
docs/wrg-demo/
├── README.md                    — This file (index)
├── flows/                       — Flow specifications and step-by-step narratives
├── requirements/                — Feature requirements and acceptance criteria
├── assets/                      — Reference images, screenshots, wireframes
└── scripts/                     — Demo speeches and presenter notes
```

## Purpose

Drop any reference material here and I will use it as context when building or modifying the WRG demo. This includes:

- Flow specs and user journeys
- Stakeholder feedback and requirements
- Figma links and design references
- Demo scripts and speaker notes
- Before/after screenshots
- Technical constraints and decisions

## How to add documentation

1. Drop files into the appropriate subfolder (`flows/`, `requirements/`, `assets/`, or `scripts/`)
2. Tell me what you added and I'll read it before making changes
3. For large context, I can summarize first then work from the summary
